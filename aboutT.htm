Summary

This website displays weather information from the suburb of Latham in the Australian Capital Territory. The weather station in use is an <#stationtype>, and these pages are updated every <#interval> minutes. The meteorological day used at this station ends at <#rollovertime>. Occasional breaks in the data do occur as this all running on an old Windows XP PC which does have a tendency to freeze at times. Apologies if this causes any issues.

The weather graphs are generated automatically by Cumulus v<#version> (<#build>)
